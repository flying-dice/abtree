# @abtree/hello-world

Greet a user based on time of day. Demonstrates sequence, selector, and action primitives.

![tree](./tree.svg)

## Run it

Paste this brief into Claude Code, ChatGPT, or any shell-capable agent:

```text
Install the npm package @abtree/hello-world, then drive the workflow:

  abtree --help
  abtree execution create ./node_modules/@abtree/hello-world "Greet me based on the current time"
```

## Install and run

See [Using a tree](https://abtree.sh/guide/using-trees) for the long-form walkthrough. `<pkg>` for this tree is `@abtree/hello-world`.
